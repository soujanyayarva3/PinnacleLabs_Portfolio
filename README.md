# PinnacleLabs_Portfolio
Portfolio Website
Personal Portfolio Website built using HTML, CSS, and JavaScript

Overview
This is a personal portfolio website that showcases my skills, projects, and contact information. It is designed to present my web development work in a clean and professional manner.

Objective
To create a responsive portfolio website to display my internship projects and web development skills.

Features
Responsive design for all devices
About section with personal introduction
Skills section showcasing technical abilities
Projects section with live links
Contact section with email and LinkedIn

Technologies Used
HTML
CSS
JavaScript

Project Workflow
Design layout using HTML structure
Style the website using CSS
Add interactivity using JavaScript
Add project links and contact details
Deploy using GitHub Pages

Result
A fully functional personal portfolio website that showcases my projects and skills in web development.
